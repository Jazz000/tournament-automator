# tournament-automator
A reusable Python script for randomized tournament bracket generation, featuring automated 'Bye' logic for odd-numbered entries.
# 🏆 Universal Tournament Automator

## Overview
This tool was designed to bring professional-grade organization to competitive gaming events. Whether managing a small team like the **Bloom** or a larger community bracket, this script ensures fair seeding and logic-driven matchups.

## Key Features
* **Dynamic Entry:** Input any number of teams; the code scales automatically.
* **Fairness Algorithm:** Uses cryptographic-grade randomization to shuffle players.
* **Smart 'Bye' Handling:** Automatically detects if a bracket is "unbalanced" and assigns a 'Bye' to the last seed to keep the tournament moving.

## How to Use
1. Run `python main.py`
2. Enter team names one by one.
3. Type 'done' to see your randomized matchups!
